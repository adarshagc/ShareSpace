# ShareSpace

ShareSpace is a Spring Boot backend application for creating and managing workspac.

## Tech Stack

--> Java
--> Spring Boot
--> MongoDB
--> Maven
--> REST API's

## Project Structure

- controller --> Handles API requests
- service --> Buisness logic
- repository --> Database interaction
- model -> Data models

## Features

-  Create Workspace API
-  MongoDB integration
-  Layered Architecture
-  RESTful API's
